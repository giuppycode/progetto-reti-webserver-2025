Il Progetto

Questo progetto è stato sviluppato come esempio didattico per mostrare come implementare un server HTTP minimale utilizzando Python e il modulo socket. L'obiettivo è fornire un esempio funzionante ma semplice.

Caratteristiche

  • Server HTTP implementato con Python e socket
  • Gestione di richieste GET
  • Risposte 200 OK e 404 Not Found
  • Servizio di file statici (HTML, CSS)
